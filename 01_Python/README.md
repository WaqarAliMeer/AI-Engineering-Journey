# 💰 Personal Expense Tracker (Python)

A simple command-line expense tracker built in Python as my first portfolio project while learning AI Engineering and Data Analytics.

This application allows users to record income and expenses, store transactions in a JSON file, and generate a financial summary.

---

## 🚀 Features

- Add income transactions
- Add expense transactions
- View all transactions
- Display financial summary
- Save transactions to a JSON file
- Menu-driven command-line interface

---

## 📂 Project Structure

```
Expense-Tracker/
│
├── main.py               # Main application
├── transactions.json     # Stores transaction data
```
